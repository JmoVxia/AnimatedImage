# AnimatedImage

High-performance animation image library.



# Features

- [x] Support playback APNG, GIF, WebP
- [x] Automatically adjust playback quality
- [x] Synchronize frame update 
- [x] Customizable playback format

# Required

- Swift 5.9
- iOS 16+
